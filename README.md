# parse_date
Build time comparisson against parse date functionality from a [blog post](https://derekrodriguez.dev/parse-dont-validate-through-the-years-with-c-/)

# Tools needed to run the tests:
- [hyperfine](https://github.com/sharkdp/hyperfine)
- cmake, make and g++ with support for C++23

# Run
`make prepare && make`
